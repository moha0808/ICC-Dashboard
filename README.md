# ICC Cricket Analytics Dashboard

An interactive Power BI dashboard designed to analyze and visualize historical ICC cricket datasets, providing deep insights into team performances, player statistics, and match outcomes.

## 📊 Project Overview

This project transforms raw ICC cricket data into actionable visual insights. Using Power BI, the dashboard allows users to explore trends, compare team/player metrics, and drill down into historical match statistics.

### Key Features
* **Team Performance Analysis:** Track wins, losses, win percentages, and trends across different eras and formats.
* **Player Leaderboards:** Visualize top run-scorers, highest wicket-takers, and strike-rate analysis.
* **Interactive Filters:** Dynamically filter data by year, team, opponent, venue, and match format.
* **Key Performance Indicators (KPIs):** High-level summaries of total matches played, boundaries scored, and milestone records.
